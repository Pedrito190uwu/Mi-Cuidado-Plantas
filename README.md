# Plant Care App - Gestión Inteligente de Plantas

## Descripción del Proyecto

**Plant Care App** es una aplicación móvil desarrollada con React Native y Expo que permite a los usuarios gestionar eficientemente el cuidado de sus plantas. La aplicación resuelve la problemática común de olvidar regar las plantas, proporcionando recordatorios inteligentes y seguimiento personalizado.

### Problemática Identificada

Muchas personas tienen plantas en sus hogares pero:
- Olvidan regarlas regularmente
- No recuerdan cuándo fue el último riego
- Desconocen las necesidades específicas de cada planta
- Pierden el seguimiento de múltiples plantas

### Solución Propuesta

La aplicación ofrece:
- **Recordatorios automáticos** de riego por planta
- **Seguimiento visual** del estado de cada planta
- **Clasificación** por ubicación (interior/exterior)
- **Guía de cuidados** específicos por especie
-  **Estadísticas** del jardín personal
